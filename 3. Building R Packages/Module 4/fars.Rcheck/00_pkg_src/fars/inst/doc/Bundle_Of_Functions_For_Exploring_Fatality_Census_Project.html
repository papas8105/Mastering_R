Fars Package Functions

library(fars)

Helper Functions To Explore The Fatality Cencus Datasets

This packages aims to produce simple graphics to show the accidents in US territory. In this vignette we are going to give simple examples of how to use the functions of the package, for this purpose we have stored three data sets found in

, we will use accident_2013, there are two other data files in . We provide some description but with some examples, you are free to experiment on your own.

The make_filename function

Creates the filename by entering the year as input.

filename <- make_filename(2013)
basename(filename)
#> [1] "accident_2013.csv.bz2"

The fars_read function

The fars_read will use as input the filename string from make_filename.
